# Ethereum Pet Shop

## Requirements
- Git
- Ganache

## Setup
1. Compile ```truffle compile```
2. Start Ganache
3. Run migrations ```truffle migrate```
4. Get npm packages ```npm install```
5. Start dapp ```npm run dev```

## Test
1. Compile ```truffle compile```
2. Start Ganache
3. Run migrations ```truffle migrate```
4. Run tests ```truffle test```
